CORS Setup Required — Read Before Fetching

▲ COLLAPSE

This app runs directly in your browser. AlienVault OTX blocks cross-origin requests from local files by default (CORS policy). You need a browser extension to inject the missing headers — takes 60 seconds.

1
Install a CORS-unblocking extension
Do not use FoxyProxy — that's a proxy router, not a CORS fix. Install one of these instead:
■ CORS Unblock (Chrome) ■ Allow CORS (Chrome) ■ CORS Everywhere (Firefox)
2
Enable the extension
Click the extension icon in your browser toolbar and toggle it ON. The extension intercepts responses from OTX and injects Access-Control-Allow-Origin: * so your browser accepts them.
3
Paste your OTX API key below and fetch
Get your key from otx.alienvault.com/api → "Your API Key". Paste it in the field below and click Fetch IOCs.
4
Turn the extension OFF after you're done
These extensions disable CORS protection globally. Only keep it on while using this tool — toggle it off before normal browsing.
⚠ Security note: CORS extensions weaken your browser's same-origin protection. Use them only on trusted pages and disable immediately after use.
